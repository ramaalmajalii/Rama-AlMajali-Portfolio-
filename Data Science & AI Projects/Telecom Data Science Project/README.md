# Telecom Data Science Project

Data science assignment for the **Data Science Programming** course (Fall 2023/2024).  
The project demonstrates a full machine learning workflow using Python for a telecom dataset containing customer information, segmentation, churn indicators, and usage patterns.

## Project Overview
A proof-of-concept analysis for a telecom company exploring:
- Customer segmentation (SEGMENT1, SEGMENT2)
- Line status prediction (STATUS)
- Churn-related patterns

The project includes:
- Data loading and cleaning
- Feature preprocessing
- Model training & evaluation
- Visualizations

## Tools & Libraries
- **Python**, **Pandas**, **NumPy**
- **Scikit-Learn** (models & metrics)
- **Matplotlib**, **Seaborn**
- **Jupyter Notebook**
