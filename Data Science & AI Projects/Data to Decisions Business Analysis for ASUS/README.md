# ASUS Laptop Sales Analytics & Forecasting

This project analyzes one year of Amazon sales data for ASUS **ROG, TUF, and Zephyrus** laptops. The dataset was cleaned and modeled using **Power BI** for visual insights and **Orange (ARIMA)** for 12-month forecasting.

## Tools Used
Power BI • Orange Data Mining • ARIMA • Data Cleaning • Time-Series Analysis
