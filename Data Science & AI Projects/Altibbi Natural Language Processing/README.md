# Medical Specialties Text Classification (NLP)

A multi-class NLP project that classifies patient questions into 5 medical specialties using traditional ML, deep learning, and transformer models.

## Pipeline
- Text cleaning & preprocessing  
- TF-IDF + embeddings  
- Model training & evaluation  

## Models
Logistic Regression • SVM • Naive Bayes • LSTM • BERT • GPT


## Tools
Python • Scikit-Learn • NLTK • TensorFlow/Keras • HuggingFace Transformers
