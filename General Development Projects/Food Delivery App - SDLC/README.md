# MomChef Homemade Bites App – SDLC Project

A full Software Development Life Cycle (SDLC) project for designing and developing **MomChef**, a food-delivery platform connecting home chefs with customers. The project follows structured SDLC stages including requirements gathering, feasibility analysis, design, development, and testing.

## Project Overview
MomChef allows home chefs to list meals, and customers to browse, order, and manage profiles. Admins manage users, chefs, and platform data.

## Main SDLC Deliverables
- **Requirements Gathering:** Background study (Talabat, Careem), interviews, functional & non-functional requirements  
- **Feasibility Study:** Technical, operational, legal, scheduling, economic  
- **System Design:** DFDs, flowcharts, ERD, process modelling  
- **Implementation:** Role-based app built using **Adalo**  
  - Customer: browse meals, cart, checkout  
  - Chef: add/edit/delete meals  
  - Admin: manage users & data  
- **Testing:** Functional testing, UI testing, data validation  
- **Evaluation:** Strengths, limitations, suggested improvements
